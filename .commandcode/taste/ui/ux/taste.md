# UI/UX
- When user requests "darker" colors in light mode, map bright hues to denser variants rather than just darkening the same hue. Confidence: 0.70
- For permission systems, implement persistent deny/never scope alongside allow grants. Confidence: 0.75
- Use 0.5px borders when user is emphatic about thin borders (e.g., "0.5px!!!!!!"). Confidence: 0.90
- In Morph theme and similar light themes, do not dim enabled options — only dim disabled/unavailable options. Confidence: 0.80
- Extensions need settings accessible via cog icon in the extensions list. Confidence: 0.75
- Allow bookmarking/pinning extensions to sidebar for quick access. Confidence: 0.75
